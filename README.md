# 造梦西游4 · 天机辅助器 v1.0

> 一个轻量级、通用的造梦西游4内存辅助工具。
> 支持 360游戏大厅 / 造梦微端 / 浏览器 / Flash独立播放器。

## 功能

| 功能 | 说明 | 原理 |
|------|------|------|
| 无敌模式 | 血量锁定在极高值，怪物打你不掉血 | 冻结血量内存地址 |
| 无限血量 | 血量始终回满 | 冻结血量地址 |
| 无限法力 | 法力始终回满，技能随便放 | 冻结法力地址 |
| 一击必杀 | 攻击力锁定在最大值，打怪秒杀 | 冻结攻击力地址 |
| 怪物自杀 | 怪物血量设为0，进图怪全死 | 扫描并冻结怪物血量 |
| 全部关闭 | 一键关闭所有作弊功能 | — |

## 使用流程

下载后按照以下步骤操作：

1. **安装 Python**（如果还没装）
   - 下载：https://www.python.org/downloads/
   - 安装时**务必勾选** "Add Python to PATH"

2. **安装依赖**
   ```
   pip install -r requirements.txt
   ```

3. **启动游戏**
   - 打开 360游戏大厅 / 造梦微端 / 浏览器
   - 进入造梦西游4，选好角色进入游戏场景

4. **启动辅助器**
   - 双击 `启动辅助器.bat`
   - 或者运行 `python main.py`

5. **连接游戏**
   - 点击「查找进程」— 自动扫描游戏进程
   - 点击「连接」— 建立内存通信

6. **扫描数值（关键步骤）**
   - **血量扫描**：看游戏里你的血量是多少，输入数字，点「首次扫描」
       然后让怪打你一下，输入新的血量值，点「再次扫描」
       重复直到只剩几个地址
   - **法力/攻击力**：同上操作

7. **开启作弊**
   - 点击对应的开关按钮即可
   - 绿色=开启，灰色=关闭

## 常见问题

**Q: 找不到游戏进程？**
A: 确保游戏已经启动并且进入了主场景。如果自动查找失败，可以手动确认进程名：
   - 打开任务管理器 → 详细信息
   - 找到游戏进程（如 360Game.exe / chrome.exe）
   - 联系我们添加该进程名

**Q: 扫描结果为0个地址？**
A: 确保输入了正确的数值（血量就是游戏里显示的数字）。
   如果值是"1234/5678"这种格式，只输入当前值 1234。

**Q: 开启功能后没效果？**
A: 可能扫描到的地址不正确。重新扫描，多重复几次"受伤害→再次扫描"，
   直到地址只剩 1~3 个再开启功能。

**Q: 会封号吗？**
A: 内存修改有一定风险。建议：
   - 不要在排行榜/PVP场景使用
   - 适可而止，不要过于夸张
   - 本工具仅供学习研究
