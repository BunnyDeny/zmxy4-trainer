# 造梦西游4 · 天机辅助器 v1.0

> 轻量级、通用的造梦西游4内存辅助工具。
> 支持 360游戏大厅 / 造梦微端 / 浏览器 / Flash独立播放器。

## 使用步骤

### 第一步：启动

1. 打开 360游戏大厅 / 造梦微端 / 浏览器，进入造梦西游4
2. 选好角色，**进入游戏场景**（在选人界面或加载页面时内存数据还没生成）
3. 双击 `zmxy4-trainer.exe` 启动辅助器

### 第二步：连接游戏进程

辅助器启动后，在左侧「连接」面板：

```
点击 [查找进程]  →  工具自动扫描系统进程
  → 找到后显示进程名和PID
  → 点击 [连接]  →  状态变为"已连接"
```

> 支持的进程名：360Game.exe / chrome.exe / msedge.exe / FlashPlayer*.exe 等

### 第三步：数值扫描（⚠️ 关键步骤）

这个辅助器使用 **Cheat Engine 式扫描**——它不知道你的血量地址在哪，需要你帮它定位。

**为什么需要扫描？**
游戏每次启动，血量、法力等数据在内存中的存放位置都会变化。所以我们需要通过"输入数值 → 扫描 → 让数值变化 → 再扫描"的方式，一步步筛出正确的地址。

#### 血量扫描流程（以血量为例）

```
 ① 看游戏里你的血量数值（比如 1850/5000）
    ↓
 ② 在左侧「血量」输入框输入 1850
    ↓
 ③ 点击 [首次扫描]  →  找到几万个地址
    ↓
 ④ 进图让怪打你一下（血量减少）
    ↓
 ⑤ 看游戏里新的血量值（比如 1520/5000）
    ↓
 ⑥ 输入 1520，点击 [再次扫描]  →  地址变少
    ↓
 ⑦ 再挨打一次 → 输入新值 → [再次扫描]
    ↓
 ⑧ 重复 ④~⑦，直到地址只剩 1~3 个
```

> 通常重复 3~4 次就能筛到只剩几个地址，越少越准。

**法力和攻击力** 的扫描流程完全相同，分别在对应的输入框操作。

### 第四步：开启作弊

地址扫描完成后（右侧面板显示地址数量），点击右侧任意功能开关：

- 点击「无敌模式」→ 按钮变绿发光 → 效果生效
- 再次点击 → 变灰 → 效果关闭
- 点击「全部关闭」→ 一键关所有

### 完整操作示例

```
[打开游戏, 进入地图]
  ↓
[启动辅助器 → 查找进程 → 连接]
  ↓
[记下血量 2000 → 输入2000 → 首次扫描 → 地址: 52341个]
  ↓
[让怪打一下 → 血量变成 1500 → 输入1500 → 再次扫描 → 地址: 328个]
  ↓
[再挨打 → 血量 1200 → 输入1200 → 再次扫描 → 地址: 6个]
  ↓
[再挨打 → 血量 900 → 输入900 → 再次扫描 → 地址: 2个]  ← 够了！
  ↓
[点击「无限血量」→ 绿了 → 生效！]
```

## 各功能详解

### 🛡️ 无敌模式
- **原理**：将血量锁定在一个极大值（9999999）
- **和无限血量的区别**：无限血量是把血量锁在你当前最大值，无敌模式锁在更高的值，且不会被打出受伤动画
- **需要**：先扫描出血量地址

### ❤️ 无限血量
- **原理**：每秒 20 次将血量地址写回最大值
- **效果**：怪打你 → 扣血 → 瞬间回满（肉眼几乎看不到掉血）
- **需要**：先扫描出血量地址

### 💧 无限法力
- **原理**：每秒 20 次将法力地址写回最大值
- **效果**：技能随便放，蓝永远用不完
- **需要**：先扫描出法力地址

### ⚔️ 一击必杀
- **原理**：将攻击力锁定在 9999999
- **效果**：一下平A直接秒杀任何怪物
- **注意**：首次使用需要先**扫描攻击力地址**——看游戏面板里的攻击数值，输入后首次扫描，然后换装备/升级改变攻击力数值，再次扫描
- **需要**：先扫描出攻击力地址

### 💀 怪物自杀
- **原理**：在玩家血量地址附近的内存区域，寻找其他处于合理血量范围（1~10000）的浮点数值，视为怪物血量，并将其写为 0
- **效果**：进图后怪物直接死亡
- **注意**：这个功能是**探索式**的——它在你血量地址周围猜测哪些是怪物的血量，不保证 100% 准确
- **需要**：先扫描出血量地址

## 常见问题

**Q: 能否做到"进图点一下秒杀就生效"，不用手动扫描？**
A: 当前版本不行，需要手动扫描定位地址。要做到一键生效，需要预先用 Cheat Engine 逆向出游戏内存的固定基址，这部分计划在后续版本实现。

**Q: 找不到游戏进程？**
A: 确保游戏已启动并进入了场景。如果自动查找失败：
   - 打开任务管理器 → 详细信息
   - 看进程名（360Game.exe / chrome.exe 等）
   - 在 GitHub Issues 里告诉我们，我们加进识别列表

**Q: 扫描结果为 0 个地址？**
A: 常见原因：
   - 还没点「连接」（先连接再扫描）
   - 输入的数值不对（只看当前值，不是"1234/5678"的完整格式）
   - 游戏还没完全加载完成

**Q: 开启功能后没效果？**
A: 说明扫描到的地址不对。重新走一遍扫描流程，多重复几次"数值变化→再次扫描"，直到只剩 1~3 个地址。

**Q: 会封号吗？**
A: 内存修改有一定风险，建议：
   - 不要在排行榜 / PVP 场景使用
   - 适度使用，不要过于夸张
   - 本工具仅供学习研究
